# Wine Quality Prediction - ML vs Deep Learning

## Overview
This project predicts wine quality (binary classification) using Machine Learning and Deep Learning.

- **Dataset:** [UCI Wine Quality Dataset](https://archive.ics.uci.edu/ml/datasets/Wine+Quality)
- **Task:** Binary classification (Good wine >=7, Not Good <7)
- **Models:**
  - Decision Tree
  - Random Forest
  - Naive Bayes
  - Neural Network
